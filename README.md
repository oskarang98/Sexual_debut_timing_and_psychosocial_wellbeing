# Sexual_debut_timing_and_psychosocial_wellbeing
This study uses data from a Norwegian population-based 28-year cohort followed from adolescence to midlife and applies inverse probability of treatment weighting to examine prospective associations between timing of sexual debut and midlife psychosocial well-being, including mental health, social health, self-evaluation, and substance use outcomes.

The repository is organized into four main scripts:

01_data_preprocessing

02_main_analysis

03_sensitivity_analysis

04_figures_and_tables
